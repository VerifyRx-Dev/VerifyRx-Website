How it works

Three steps between a patient
and a verified medicine

Our end-to-end system connects manufacturers, pharmacists, and patients in one seamless verification chain.

01

🏭

Manufacturer registers

A pharmaceutical company joins VerifyRx and generates a batch of cryptographically signed QR codes — one unique code per drug pack.

02

📦

Codes applied to packs

Each QR code is printed on the packaging at the factory. The code is signed with HMAC-SHA256, making it impossible to forge or duplicate.

03

📱

Patient scans & verifies

Anyone with a smartphone scans the QR code. The result is instant — genuine drug with product details, or a counterfeit warning. Each code works only once.

Why VerifyRx

Built to stop counterfeits at every level

Every feature is designed with one goal: making it impossible for a fake drug to reach a patient undetected.

🔒

One-time scan lock

Each QR code is invalidated the moment it is scanned. If someone tries to scan a copied code, the system immediately flags it as a duplicate.

📡

Offline partial verification

The HMAC signature embedded in every code allows basic authenticity checks even without internet — critical for rural areas.

📍

Geo-fraud intelligence

Every scan is logged with GPS coordinates. Our system clusters suspicious scan patterns graphically and alerts manufacturers automatically.
